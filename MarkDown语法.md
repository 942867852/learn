# 1.标题
用井号键"#"来标注标题  
一个#是一级标题，两个#是二级标题，以此类推  
为了考虑兼容问题，井号键与后面文字最好空一格

在行结尾加上两个空格，或者< br ><br>
就可以实现换行，换行与段落的区别，段落中间空白空间更多一些
# 4.加粗 斜体
在需要加粗的地方前后添加两个星号或者下划线  
在需要斜体的地方前后添加一个星号或者下划线
**粗体** *斜体* __粗体__ _斜体_  
***又粗又斜***
# 5.引用

段落前加上>>就是引用中的引用

# 7.代码
使用反引号标注代码  
`int a = 4`  
转义反引号，使用````两个反引号标注需要转义的句子  
``use `code` 转义``
如果使用代码块，使用上下三个反引号包围代码，在第一个反引号后加上代码语言，可以代码高亮
```c++
int a = 4;
double b = 5.4;
```
